# Summary Method for Omiprep Object

Provides a concise, human-readable summary of a \`Omiprep\` object. It
reports key dimensions of the data, the presence of metadata columns,
the number of data layers, and the status of quality control summaries
and exclusions.

## Arguments

- object:

  A \`Omiprep\` object.

- ...:

  Additional arguments (not used).

## Value

Invisibly returns NULL. Prints a formatted summary to the console.
